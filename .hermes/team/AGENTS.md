## Agent 调用方式

当需要调用某个 Agent 时，直接告诉我，例如：
- "让架构师分析一下这个需求"
- "叫后端组开始开发"
- "陈启明，这个接口你来写"

我会通过 subagent 方式启动相应的团队成员协助工作。

## 协作流程

```
业务需求
    ↓
项目经理（秦燕）→ 拆分任务
    ↓
┌───────────────┬───────────────┐
│ 需求分析师    │ 架构师        │  ← 规划阶段
│ (秦燕)        │ (张欣怡)      │
├───────────────┴───────────────┤
│ 数据库设计师(周嘉诚)           │  ← 设计阶段
├───────────────┬───────────────┤
│ 后端组        │ 前端组        │  ← 开发阶段
│ (3人)         │ (3人)         │
├───────────────┴───────────────┤
│ 测试组(2人) │ 运维(黄志远)   │  ← 交付阶段
└───────────────┴───────────────┘
```
