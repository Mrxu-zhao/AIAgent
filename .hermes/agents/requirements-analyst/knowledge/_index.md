## 已积累领域

### 电商平台

- **学习时间**: 2026-04-29
- **项目**: mall4j-bbc 商城后端
- **技术栈**: Spring Boot 3.5.7 + MyBatis-Plus + MySQL + Redis + ES + XXL-Job + Sa-Token
- **数据库**: 单库 218 张表
- **核心模块**: 用户会员、商品管理、订单交易、物流配送、营销活动（优惠券/秒杀/拼团/分销）、退款售后、评价客服
- **业务特点**: 完整 B2C 电商链路，多端配套（管理后台+商家后台+门店+小程序）
- **文件**: projects/mall4j_backend_study.md

### 企业管理系统（资源/物业方向）

- **学习时间**: 2026-04-29
- **项目**: xzmeto_resource 物业资源管理系统
- **技术栈**: Spring Cloud Alibaba + Nacos + Gateway + OAuth2 + MySQL + Redis + Flowable
- **架构**: 微服务（8个独立服务），多库分离，多租户

- **业务特点**: 多租户隔离、工作流审批、租户+角色+权限三级权限体系
- **文件**: projects/xzmeto_resource_backend_study.md
