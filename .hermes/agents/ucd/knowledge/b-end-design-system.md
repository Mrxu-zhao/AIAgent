# B端产品 UI 设计系统规范

## 概述

本文档定义企业管理系统（B端产品）的UI设计系统规范，适用于使用 Vue 3 + Element Plus 技术栈的团队。

## 设计原则

### 1. 一致性原则
- 相同功能在不同页面表现一致
- 遵循 Element Plus 默认设计语言
- 保持视觉风格统一（色彩、字体、间距）

### 2. 效率优先原则
- 减少用户操作步骤
- 提供快捷操作和批量处理
- 支持键盘快捷键

### 3. 可控性原则
- 用户操作可逆（撤销/重做）
- 明确的状态反馈
- 危险操作二次确认

| H2 | 20px | 28px | 模块标题 |


## 间距系统

### 基础间距单位：4px

| 名称 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 紧凑元素间距 |
| sm | 8px | 组件内部间距 |
| md | 16px | 组件之间间距 |
| lg | 24px | 模块之间间距 |
| xl | 32px | 页面区域间距 |

## 组件规范

### 按钮
```
尺寸：large(40px) / default(32px) / small(24px)
圆角：4px（统一）
```

按钮优先级：
1. 主按钮：一个页面只用一个主要操作
2. 默认按钮：次要操作
3. 文字按钮：辅助操作、链接
4. 危险按钮：删除、禁用等危险操作

### 表格
```
行高：48px（数据行）/ 56px（表头）
边框：1px solid #EBEEF5
斑马纹：#FAFBFC（可选）
```

### 表单
```
标签宽度：120px（右对齐）
输入框高度：32px
必填标识：红色 * 号
```

## 页面布局

### 标准后台布局
```
┌─────────────────────────────────────┐
│ 顶部导航栏 (Header)    高度：56px   │
├──────────┬──────────────────────────┤
│          │                          │
│ 侧边导航  │     主内容区域           │
│ 宽度：    │     (Main Content)      │
│ 200px    │                          │
│ 可折叠    │     padding: 24px        │
│          │                          │
└──────────┴──────────────────────────┘
```

### 响应式断点
| 设备 | 宽度 |
|------|------|
| 大屏 | ≥1920px |
| 桌面 | ≥1200px |
| 平板 | ≥768px |
| 手机 | <768px |

## 图标规范

使用 Element Plus 内置图标或 IconPark：
- 图标尺寸：16px（小）/ 20px（中）/ 24px（大）
- 图标颜色：跟随文字颜色
- 操作图标放在操作列右侧

## 无障碍设计

- 颜色对比度 ≥ 4.5:1
- 支持 Tab 键焦点导航
- 表单标签必须存在
- 图片提供 alt 文字
