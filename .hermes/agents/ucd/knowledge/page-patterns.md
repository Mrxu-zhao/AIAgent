# 管理后台页面模式规范

## 一、列表页模式

### 标准结构
```
┌─────────────────────────────────────────────────┐
│ 页面标题                    [新增] [批量操作]   │
├─────────────────────────────────────────────────┤
│ [筛选区域]                                       │
│ 搜索框 | 下拉选择 | 日期范围 | [重置] [搜索]    │
├─────────────────────────────────────────────────┤
│ [数据表格]                                       │
│ ┌──┬──────┬───────┬──────┬─────────┬─────────┐  │
│ │☑ │编号  │名称   │状态  │创建时间 │操作    │  │
│ ├──┼──────┼───────┼──────┼─────────┼─────────┤  │
│ │☐ │001   │张三   │启用  │2024-01  │[详情][编辑][删除]│
│ └──┴──────┴───────┴──────┴─────────┴─────────┘  │
├─────────────────────────────────────────────────┤
│ 共 128 条  [◀ 1 2 3 ... 13 ▶]  每页 10条       │
└─────────────────────────────────────────────────┘
```

### 筛选区域规范
- 筛选条件 ≤ 3 行
- 超过 3 行使用「更多筛选」
- 必填筛选置顶显示
- 重置按钮清空所有筛选条件

### 表格规范
| 元素 | 规范 |
|------|------|
| 复选框列 | 宽度 40px，固定左侧 |
| 操作列 | 宽度 160px，固定右侧 |
| 文字列 | 左对齐，超长省略号 |
| 数字列 | 右对齐 |
| 时间列 | 宽度 160px，格式：YYYY-MM-DD HH:mm |

### 空状态
- 显示空状态插图
- 提示文字：「暂无数据」
- 提供操作入口（如「去新增」）

## 二、表单页模式

### 标准结构
```
┌─────────────────────────────────────────────────┐
│ [返回]  表单标题                                 │
├─────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────┐ │
│ │ 基础信息                                    │ │
│ │ ─────────────────────────────────────────── │ │
│ │ 用户名 *    [____________]                   │ │
│ │ 手机号      [____________]                   │ │
│ │ 邮箱        [____________]                   │ │
│ └─────────────────────────────────────────────┘ │
│ ┌─────────────────────────────────────────────┐ │
│ │ 角色配置                                    │ │
│ │ ─────────────────────────────────────────── │ │
│ │ 角色      [☐ 管理员][☑ 编辑][☐ 查看]       │ │
│ │ 部门      [____________▼]                   │ │
│ └─────────────────────────────────────────────┘ │
│                    [取消]  [保存]                │
└─────────────────────────────────────────────────┘
```

### 布局规范
| 类型 | 说明 |
|------|------|
| 单列布局 | 宽度 60%，适用简单表单 |
| 双列布局 | 适用字段数较多表单 |
| 卡片分组 | 字段按业务分组 |

### 标签对齐
- 右对齐：适合快速浏览
- 左对齐：适合阅读理解
- 标签宽度：固定 120px

### 必填标识
- 使用红色 * 号
- 不标注「必填」文字
- 选填字段标注「选填」

## 三、详情页模式

### 标准结构
```
┌─────────────────────────────────────────────────┐
│ [返回]  用户详情           [编辑] [删除]         │
├─────────────────────────────────────────────────┤
│ ┌───────────────────┬─────────────────────────┐ │
│ │ 基本信息           │ 其他信息                 │ │
│ │ ──────────────── │ ──────────────────────── │ │
│ │ 姓名：李四        │ 创建人：管理员          │ │
│ │ 手机：138****8888 │ 创建时间：2024-01-15   │ │
│ │ 邮箱：li@xx.com   │ 更新时间：2024-02-01   │ │
│ └───────────────────┴─────────────────────────┘ │
│ ┌─────────────────────────────────────────────┐ │
│ │ 操作日志                                     │ │
│ │ ─────────────────────────────────────────── │ │
│ │ 2024-02-01 10:00  修改了角色权限    张三    │ │
│ │ 2024-01-15 14:30  创建了用户        李四    │ │
│ └─────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────┘
```

### 信息展示规范
- 标签与内容用冒号分隔
- 敏感信息脱敏显示
- 关键信息突出显示
- 相关联信息可点击跳转

## 四、仪表盘模式

### 标准结构
```
┌─────────────────────────────────────────────────┐
│  数据概览                                        │
│ ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐   │
│ │ 总用户 │ │ 活跃   │ │ 新增   │ │ 转化率 │   │
│ │  10,234│ │  5,678 │ │   321 │ │  85.2% │   │
│ │ ↑12%  │ │ ↑5.3%  │ │ ↓2.1% │ │ ↑0.8% │   │
│ └────────┘ └────────┘ └────────┘ └────────┘   │
├─────────────────────────────────────────────────┤
│ ┌─────────────────────┐ ┌─────────────────────┐ │
│ │ 趋势图              │ │ 分布图              │ │
│ │ [折线图]           │ │ [饼图]              │ │
│ └─────────────────────┘ └─────────────────────┘ │
│ ┌─────────────────────┐ ┌─────────────────────┐ │
│ │ 排行榜              │ │ 快捷入口            │ │
│ │ 1. 区域A   1,234    │ │ [用户管理]          │ │
│ │ 2. 区域B     987    │ │ [订单管理]          │ │
│ └─────────────────────┘ └─────────────────────┘ │
└─────────────────────────────────────────────────┘
```

### 指标卡规范
- 数字突出显示（大字号）
- 显示同比/环比变化
- 提供趋势指示器
- 点击可跳转详情

### 图表规范
- 图表高度：300-400px
- 图例位置：底部或右侧
- 数据过多时显示前5+其他
- 支持时间范围切换

## 五、导航设计规范

### 顶部导航
- 高度：56px
- Logo 居左
- 导航菜单水平排列
- 用户信息居右

### 侧边导航
- 宽度：200px（展开）/ 64px（折叠）
- 一级菜单最多 8 个
- 二级菜单最多 10 个
- 当前选中项高亮显示
- 支持多级菜单（最多3级）

### 面包屑
- 格式：首页 > 模块 > 页面
- 当前位置高亮
- 支持点击跳转

## 六、通用交互规范

### 加载状态
- 页面加载：骨架屏
- 局部加载：Loading 动画
- 按钮加载：显示加载图标+禁用

### 操作反馈
| 操作 | 反馈方式 |
|------|----------|
| 保存成功 | Toast 提示 + 关闭弹窗/跳转 |
| 保存失败 | 错误提示在表单顶部 |
| 删除确认 | 模态框确认 |
| 批量操作 | 进度条或 Loading |

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| Enter | 确认/提交 |
| Esc | 取消/关闭 |
| Ctrl+S | 保存 |
| Ctrl+K | 全局搜索 |
