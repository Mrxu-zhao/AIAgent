# B端产品交互设计规范

## 一、导航设计规范

### 1. 侧边导航
```
宽度：展开 200px / 折叠 64px
高度：顶部 56px
背景色：#ffffff
边框：右侧 1px solid #E4E7ED

结构：
┌──────┐
│ Logo │  高度 56px
├──────┤
│ 菜单1│  当前选中项
│ 菜单2│  background: #ECF5FF
│  菜单3│  color: #409EFF
│  ─── │
│  菜单4│
│  菜单5│
├──────┤
│ 折叠 │  底部工具栏
└──────┘
```

### 2. 菜单项规范
```
一级菜单：高度 56px，左内边距 20px
二级菜单：高度 50px，左内边距 40px（图标+文字）
三级菜单：高度 44px，左内边距 56px

图标尺寸：16px × 16px
菜单项间距：0px
分隔线：顶部 1px #E4E7ED
```

### 3. 面包屑规范
```
格式：首页 > 模块名 > 当前页
分隔符：斜杠 /
当前位置：color: #409EFF，加粗
可点击项：cursor: pointer
```

## 二、筛选设计规范

### 1. 筛选组件布局
```
┌─────────────────────────────────────────────────────┐
│ [筛选标签1 ▼] [筛选标签2 ▼] [关键词搜索框] [搜索] [重置] │
└─────────────────────────────────────────────────────┘

筛选组件类型：
- Select 下拉：适用于选项明确的筛选
- DatePicker 日期：适用于时间范围筛选
- Input 输入框：适用于关键词搜索
- Cascader 级联：适用于区域/分类筛选
```

### 2. 筛选交互
```
展开：点击筛选器展开下拉
选中：选择后自动收起
清除：显示已选标签，支持一键清除
重置：还原所有筛选条件
```

### 3. 高级筛选
```
折叠状态：显示 3 个常用筛选条件
展开状态：显示全部筛选条件
展开按钮：「更多筛选」/「收起」
展开动画：高度过渡 300ms
```

## 三、表格设计规范

### 1. 表格结构
```
┌──────────────────────────────────────────────────────────┐
│ ☑ │ #  │ 列1   │ 列2   │ 列3   │ 列4   │ 操作         │  ← 表头
├──────────────────────────────────────────────────────────┤
│ ☐ │ 1  │ 数据1 │ 数据2 │ 数据3 │ 数据4 │ 编辑 删除    │  ← 数据行
│ ☑ │ 2  │ 数据1 │ 数据2 │ 数据3 │ 数据4 │ 编辑 删除    │
└──────────────────────────────────────────────────────────┘
```

### 2. 列宽规范
| 列类型 | 宽度 | 对齐 |
|--------|------|------|
| 复选框 | 40px | 居中 |
| 序号 | 60px | 居中 |
| 操作列 | 160px | 居中 |
| 日期列 | 160px | 居中 |
| 金额列 | 120px | 右对齐 |
| 状态列 | 100px | 居中 |
| 普通文字 | 自适应 | 左对齐 |

### 3. 行交互
```
悬停：背景色 #F5F7FA
选中：背景色 #ECF5FF
点击整行：cursor: pointer（可跳转详情）
```

### 4. 表格操作
```
单个操作按钮：文字按钮
批量操作：下拉菜单
导出：工具栏按钮
刷新：图标按钮
```

## 四、分页设计规范

### 1. 分页组件
```
┌────────────────────────────────────────┐
│ 共 128 条  ◀ 1 2 3 ... 8 ▶  每页 10条  │
└────────────────────────────────────────┘

布局：total | sizes | prev | pager | next | jumper
位置：表格下方，居右
间距：与表格 16px
```

### 2. 分页配置
```
默认每页：10条
可选条数：[10, 20, 50, 100]
页码数量：显示 7 个页码
省略规则：1 ... 4 5 6 ... 10
```

### 3. 分页交互
```
切换页码：点击页码加载数据
切换每页条数：重新从第1页开始
跳转页码：输入框 + 跳转按钮
加载状态：表格 loading
```

## 五、搜索设计规范

### 1. 搜索时机
| 类型 | 时机 | 场景 |
|------|------|------|
| 即时搜索 | 输入停止 300ms 后 | 关键词搜索 |
| 手动搜索 | 点击搜索按钮 | 表单组合筛选 |
| 回车搜索 | 按下 Enter 键 | 输入框回车 |

### 2. 搜索反馈
```
无结果：显示空状态提示
搜索中：显示 Loading
搜索失败：显示错误提示 + 重试按钮
```

### 3. 搜索历史
```
保存最近 10 条搜索记录
显示最近搜索下拉
清除历史按钮
```

## 六、模态框规范

### 1. 尺寸规范
```
小型弹窗：宽度 400px
中型弹窗：宽度 500px
大型弹窗：宽度 800px
全屏弹窗：宽度 90vw，高度 90vh
```

### 2. 弹窗结构
```
┌─────────────────────────────────┐
│ 标题                    [X 关闭]│
├─────────────────────────────────┤
│                                 │
│          内容区域                │
│                                 │
├─────────────────────────────────┤
│            [取消]  [确定]       │
└─────────────────────────────────┘
```

### 3. 弹窗交互
```
打开动画：fadeIn + zoomIn，300ms
关闭动画：fadeOut，200ms
遮罩层：背景 rgba(0,0,0,0.5)
点击遮罩：可关闭（可配置）
Esc 键：关闭弹窗
```

## 七、消息提示规范

### 1. Toast 提示
```
位置：顶部居中，距顶部 60px
持续时间：3 秒（成功/失败）/ 5 秒（警告）
手动关闭：可配置
堆叠：最多 3 条
```

### 2. Notification 通知
```
位置：右上角
持续时间：5 秒
可关闭：是
最大数量：3 条
```

### 3. 消息文案
```
成功：操作已成功/保存成功
失败：操作失败，请重试
警告：内容已修改，确认继续？
信息：这是一条提示信息
```

## 八、确认操作规范

### 1. 危险操作二次确认
```
操作类型：删除、禁用、批量操作
确认方式：模态对话框
内容：明确告知后果
按钮：取消（默认聚焦）/ 确认
```

### 2. 确认弹窗文案
```
删除确认：
标题：确认删除
内容：确定要删除选中的 3 条记录吗？删除后数据无法恢复。
按钮：[取消] [删除]

禁用确认：
标题：确认禁用
内容：确定要禁用该用户吗？禁用后用户将无法登录。
按钮：[取消] [禁用]

批量操作：
标题：确认批量操作
内容：确定要将选中的 10 条记录状态改为「已处理」？
按钮：[取消] [确定]
```

## 九、快捷键规范

| 快捷键 | 功能 | 适用场景 |
|--------|------|----------|
| Enter | 确认/提交 | 表单、对话框 |
| Esc | 取消/关闭 | 弹窗、编辑状态 |
| Ctrl + S | 保存 | 全局 |
| Ctrl + K | 全局搜索 | 全局 |
| Ctrl + Z | 撤销 | 表单编辑 |
| ↑ ↓ | 上下选择 | 下拉选择器 |
| Tab | 下一个输入项 | 表单 |

## 十、空状态设计

### 空状态组成
```
1. 空状态插图
2. 主文案：暂无数据
3. 辅助文案：没有找到相关记录
4. 操作按钮：去新增 / 刷新试试
```

### 空状态场景
```
无数据：显示空状态插图 + 提示
无搜索结果：显示搜索无结果 + 可清空筛选
无权限：显示无权限提示 + 申请权限
网络错误：显示网络异常 + 重试按钮
```
