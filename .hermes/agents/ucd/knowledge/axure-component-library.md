# Axure 原型设计规范

## 一、组件库建设

### 1. 基础组件库结构
```
项目库
├── 1.基础组件
│   ├── 按钮（Button）
│   │   ├── 主按钮
│   │   ├── 默认按钮
│   │   ├── 文字按钮
│   │   ├── 危险按钮
│   │   └── 按钮组
│   ├── 输入框（Input）
│   │   ├── 单行输入
│   │   ├── 多行输入
│   │   ├── 带图标的输入
│   │   └── 带清除的输入
│   ├── 选择器（Select）
│   ├── 日期选择器
│   └── 开关
├── 2.容器组件
│   ├── 卡片（Card）
│   ├── 面板（Panel）
│   ├── 折叠面板
│   └── 标签页（Tabs）
├── 3.数据展示
│   ├── 表格（Table）
│   ├── 列表
│   ├── 树形控件
│   └── 分页
├── 4.反馈组件
│   ├── 弹窗（Dialog）
│   ├── 抽屉（Drawer）
│   ├── 消息提示
│   └── 确认对话框
├── 5.导航组件
│   ├── 顶部导航
│   ├── 侧边导航
│   ├── 面包屑
│   └── 标签页
└── 6.业务组件
    ├── 搜索栏
    ├── 筛选栏
    ├── 数据卡片
    └── 操作栏
```

### 2. 组件命名规范
```
命名格式：[类型]_[功能]_[状态]
示例：
- btn_primary_default      主按钮-默认
- btn_primary_hover        主按钮-悬停
- btn_primary_disabled     主按钮-禁用
- input_text_default       文本输入-默认
- dialog_confirm_delete    确认删除弹窗
```

### 3. 组件状态管理
每个可交互组件应包含以下状态：
| 状态 | 说明 |
|------|------|
| Default | 默认状态 |
| Hover | 鼠标悬停 |
| Active/Pressed | 按下状态 |
| Focus | 聚焦状态 |
| Disabled | 禁用状态 |
| Loading | 加载状态 |
| Selected | 选中状态 |
| Error | 错误状态 |

## 二、样式规范

### 1. 尺寸规范
```
按钮：
- 大按钮：高度 40px，宽度自适应
- 默认按钮：高度 32px，宽度自适应
- 小按钮：高度 24px，宽度自适应

输入框：
- 高度：32px

表格：
- 行高：48px
- 表头高：56px
- 单元格内边距：12px
```

## 三、原型标注规范

### 1. 交互标注内容
```
1. 点击操作
   - 点击对象
   - 触发动作（弹窗/跳转/展开等）
   - 目标位置

2. 状态变化
   - 触发条件
   - 变化前后对比

3. 数据传递
   - 从哪获取数据
   - 传递什么数据
   - 传递给谁

4. 业务规则
   - 校验规则
   - 权限要求
   - 数据限制
```

### 2. 标注模板
```
【标注编号】：A-001
【页面名称】：用户列表页
【组件】：表格操作列
【交互类型】：点击按钮
【交互描述】：点击"编辑"按钮，打开编辑用户弹窗
【前置条件】：用户状态为"启用"
【后置动作】：弹窗打开，数据回显
【异常处理】：无
【设计说明】：编辑后列表自动刷新
```

### 3. Axure 标注方法
```
使用方式：
1. 使用动态面板封装不同状态
2. 使用母版实现通用布局
3. 使用变量存储交互数据
4. 使用注释功能添加标注
5. 使用页面说明添加全局说明

快捷键：
- F5：预览
- Ctrl+R：生成原型
- Ctrl+Shift+O：添加说明
```

## 四、母版使用技巧

### 1. 母版分类
```
全局母版：
- 页眉（Header）
- 页脚（Footer）
- 全局导航

模块母版：
- 列表页框架
- 表单页框架
- 详情页框架

业务母版：
- 用户头像
- 状态标签
- 操作按钮组
```

### 2. 母版参数化
```javascript
// 设置母版参数
[[LVAR1]] - 占位符
[[Item.__Config__.id]] - 表格行ID

// 动态面板状态引用
[[PanelName.state]]
[[PanelName.state1]]
```

### 3. 母版管理
- 命名规范：`母版类型_版本号`
- 更新时选择「重新加载所有母版实例」
- 重要母版添加说明文档

## 五、Axure 输出规范

### 1. 文件结构
```
项目文件夹
├── 项目名.rp                  主文件
├── 低保真原型
│   ├── 页面1
│   ├── 页面2
│   └── ...
├── 高保真原型
│   └── ...
├── 标注文档
│   ├── 交互说明.docx
│   ├── 流程图
│   └── ...
└── 资源文件
    ├── 图片素材
    └── 图标库
```

### 2. 页面命名规范
```
命名格式：[模块]-[页面类型]-[序号]
示例：
- 用户-列表-001
- 用户-新增-002
- 用户-编辑-003
- 权限-分配-001
```

## 六、与开发协作

### 1. 交接内容
```
1. 原型文件

2. 设计规范
   - 色彩规范
   - 字体规范
   - 间距规范
   - 组件规范

3. 交互说明
   - 页面流程图

   - 状态说明文档
```
